# Pythominos
Puzzle game under development, school project at the origin

## Versions disponibles

### Version Pyxel (Version originale)
- **Fichier** : `Pythominos.py`
- **Plateforme** : PC (Windows, Mac, Linux)
- **Bibliothèque** : Pyxel
- Jeu complet avec musique, animations, menus, et modes de jeu variés

### Version NumWorks (Nouvelle!)
- **Fichier** : `pythominos_numworks.py`
- **Plateforme** : Calculatrice NumWorks
- **Bibliothèques** : kandinsky, ion
- Version simplifiée et optimisée pour la calculatrice
- Voir [README_NUMWORKS.md](README_NUMWORKS.md) pour plus de détails
- Voir [INSTALLATION_NUMWORKS.md](INSTALLATION_NUMWORKS.md) pour l'installation

## Installation rapide

### Version PC (Pyxel)
```bash
pip install pyxel
python Pythominos.py
```

### Version NumWorks
Consultez le guide complet : [INSTALLATION_NUMWORKS.md](INSTALLATION_NUMWORKS.md)
